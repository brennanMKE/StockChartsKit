## Description

Provide the default `portfolioHistory(accountID:range:)` implementation as a protocol extension on `BrokerageProvider` (PRD §7) so providers without native portfolio history get reconstruction for free from their injected `snapshotStore` + `marketData`.

## Scope

- `extension BrokerageProvider { public func portfolioHistory(accountID:range:) async throws -> PerformanceSeries }`.
- Throw `BrokerageError.unsupported(method: "portfolioHistory")` when both `snapshotStore` and `marketData` are nil.
- Algorithm: (1) pull snapshots in range from the store; (2) if sparse, interpolate by replaying positions × market prices — **symbols that don't resolve in market data are carried at their last known snapshot value, never dropped**; (3) resample to `range.granularity`; (4) return `PerformanceSeries`.
- Providers with `.nativePortfolioHistory` (none in v1) override this directly.
