## Description

Implement a small reusable `HTTPClient` actor (PRD §11) that every provider wraps. No third-party HTTP client — `URLSession` with `data(for:)` only.

## Scope

- `actor HTTPClient` over `URLSession`.
- Single interception point for Bearer / OAuth signing (providers inject their signer).
- Retry-with-backoff on 429 and 5xx: max 3 attempts, exponential backoff; parse `Retry-After` header and surface `BrokerageError.rateLimited(retryAfter:)` when exhausted.
- Configured `JSONDecoder`: snake_case → camelCase, ISO 8601 dates, `Decimal` from string.
- Cancellation-aware: `Task.checkCancellation()` at entry; URLSession tasks cancelled via structured concurrency.
- Map transport/decoding failures to `BrokerageError.network` / `.decodingFailed`.

## Acceptance criteria

- Tests via a `URLProtocol` stub (#0024): retry fires on 429 then succeeds; `Retry-After` parsed; decoding of snake_case + Decimal-from-string works; cancellation propagates.
- `swift build` + `swift test` pass with no live network.
