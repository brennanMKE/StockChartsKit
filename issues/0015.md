## Description

Implement the `StockChartsKitTesting` library target (PRD §12) providing test doubles used across the core and provider test suites. Landing this early unblocks tests for #0010, #0013, and others.

## Scope

- `MockBrokerageProvider`: a programmable `BrokerageProvider` conformance whose responses (accounts, positions, balances, quotes, histories, auth status) are configurable per test.
- `InMemorySnapshotStore`: an in-memory `SnapshotStore` for unit tests.
- `FixedMarketDataProvider`: returns canned `Quote` / `PerformanceSeries`.

## Acceptance criteria

- All three conform to their protocols and are `Sendable`-clean under strict concurrency.
- Smoke tests exercising each double.
- `swift build` + `swift test` pass.
