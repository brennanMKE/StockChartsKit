## Description

Implement `PortfolioAggregator` in `Sources/StockChartsKit/Aggregation/PortfolioAggregator.swift` (PRD §8) — aggregates accounts, total value, and combined history across multiple providers, doing FX conversion when summing across currencies.

## Scope

- `actor PortfolioAggregator`, `init(providers:snapshotStore:marketData:fx:)` with `fx` defaulting to `ECBFXService()`.
- `allAccounts() async throws -> [Account]` — union across providers.
- `totalValue(in:) async throws -> Money` — sum balances, FX-converting via injected `FXService`.
- `combinedHistory(range:in:) async throws -> PerformanceSeries` — merge per-account/portfolio histories into one series in the target currency.
- `liveTotalValues(in:) -> AsyncThrowingStream<Money, Error>` — streams partial totals as providers finish loading, so the macOS app can show partial results while slow providers (Fidelity) catch up.

## Acceptance criteria

- Tests using `MockBrokerageProvider` (#0015), `InMemorySnapshotStore`, `FixedMarketDataProvider`, and a stub `FXService`: total across two currencies converts correctly; `liveTotalValues` emits incremental updates and finishes; combined history aligns timestamps.
- Depends on #0012, #0015. `swift build` + `swift test` pass.
