## Description

Define `MarketDataProvider` in `Sources/StockChartsKit/Protocols/MarketDataProvider.swift` (PRD §6). The host supplies (or defaults to) a market data source for providers that lack native charting history.

## Scope

- `protocol MarketDataProvider: Sendable` with `quote(symbol:) async throws -> Quote` and `priceHistory(symbol:range:) async throws -> PerformanceSeries`.
- Depends on `Quote`, `Symbol`, `PerformanceSeries`, `TimeRange` (#0003, #0004).
