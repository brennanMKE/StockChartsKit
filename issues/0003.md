## Description

Implement `TimeRange` and `Granularity` (PRD §3.2) in `Sources/StockChartsKit/Models/TimeRange.swift`. `TimeRange` mirrors the Robinhood chart selector exactly and drives every chart query.

## Scope

- `TimeRange: String, CaseIterable, Sendable, Codable` with cases `oneDay`=`1D`, `oneWeek`=`1W`, `oneMonth`=`1M`, `threeMonths`=`3M`, `yearToDate`=`YTD`, `oneYear`=`1Y`, `fiveYears`=`5Y`, `max`=`MAX`.
- `func interval(now: Date = .now, calendar: Calendar = .current) -> DateInterval`. For `max`, returns `(.distantPast, now)` (provider clamps to available history). YTD anchors at Jan 1 of `now`'s year.
- `var granularity: Granularity` — recommended sample hint (e.g. 1-minute for `1D`, daily for ≥`1M`).
- `Granularity: Sendable, Codable` with `oneMinute`, `fiveMinutes`, `fifteenMinutes`, `oneHour`, `oneDay`, `oneWeek`.

## Acceptance criteria

- Property-based / parameterized tests covering interval rollover edge cases: **DST transitions, year boundary, leap day** (PRD §12). Verify YTD computes Jan 1 correctly across years.
- `swift build` + `swift test` pass.
