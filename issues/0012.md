## Description

Implement the `FXService` protocol and its default concrete `ECBFXService` (PRD §8). Used by `PortfolioAggregator` to convert when summing across currencies.

## Scope

- `protocol FXService: Sendable` with `rate(from:to:at:) async throws -> Decimal`.
- `ECBFXService`: hits the European Central Bank reference-rates endpoint (free, no auth) for fiat. For crypto, falls back to Coinbase's spot-price endpoint.
- Host may inject any other `FXService`.
- Network access through the shared `HTTPClient` (#0014) where practical; cache rates reasonably.

## Acceptance criteria

- Tests via HTTP replay fixtures (#0024) — no live network in `swift test`.
- Same-currency rate returns 1; cross-currency conversion produces expected `Decimal`.
- `swift build` + `swift test` pass.
